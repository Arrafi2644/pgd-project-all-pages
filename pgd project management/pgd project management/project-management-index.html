Course Content

Our Postgraduate Diploma in Project Management programme is designed to provide students with the knowledge and skills they need to succeed as project management professionals. Our curriculum covers different areas of project management, from project planning and risk management to project execution and project monitoring.

UNIT 1: PROJECT MANAGEMENT FUNDAMENTAL
Session 1: Modern Project Management
Session 2: Organisation Strategy and Project Selection
Session 3: Organising Project Structure and Culture
Session 4: Defining the Project Scope
UNIT 2: PROJECT PLANNING AND EXECUTION
Session 1: Project Initiation
Session 2: Planning for Implementation
Session 3: Implementing Projects
Session 4: Project Transition and Closure
UNIT 3: MANAGING RESOURCES IN PROJECT
Session 1: Stakeholder Management
Session 2: Managing Project Communications
Session 3: Leading and Managing Project
Session 4: Managing Resources Ethically
UNIT 4: PROJECT FEASIBILITY AND FINANCING
Session 1: Feasibility Analysis
Session 2: Financial Estimates and Projections
Session 3: Project Evaluation
Session 4: Project Financing
UNIT 5: PROJECT RISK MANAGEMENT
Session 1: Project Risk Management Systems
Session 2: Project Risk Identification
Session 3: Project Risk Analysis and Evaluation
Session 4: Risk Response, Monitoring and Control
UNIT 6: PERFORMANCE MEASUREMENT AND CONTROL
Session 1: Project Supply Chain Management
Session 2: Managing change through project management
Session 3: Monitoring and Evaluation
Session 4: Quality Management and Auditing Projects
